# Terminology

Consistent naming conventions for drift.observer.

## Core Terms

| Term | Definition | Notes |
|------|------------|-------|
| **Player** | The human playing the game | Never "user" |
| **Entity** | The consciousness the player communicates with | Use until name revealed |
| **Leandra** | The Entity's name | Revealed mid-game via [[Bootstrap Paradox]] |
| **The Correction** | RSVP intrusion phenomenon | Not "hostile" - ambiguous |

## Technical Terms

| Term | Definition |
|------|------------|
| **The Crack** | The unstable channel between Entity and player |
| **Coherence** | The Entity's stability/presence, built through dialogue |
| **RSVP** | Rapid Serial Visual Presentation - how The Correction manifests |
